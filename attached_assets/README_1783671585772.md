# uniHub brand package (v2 — brand board edition)

## Masters
- `unihub-mark.svg` — primary mark (glossy petals, dark core, red U)
- `unihub-mark-outline.svg` — white outline variant for red backgrounds

## Color palette
- Primary red     #E53935
- Deep red        #C62828
- Background dark #1A1A1A
- Surface grey    #333333
- White           #FFFFFF

## Typography
- Poppins SemiBold (wordmark), Poppins Medium (tagline, letterspaced caps)
- Tagline: CONNECT. FOCUS. ACHIEVE.

## Contents
- Favicons: `favicon.ico`, `unihub-mark-16…1024.png`
- `apple-touch-icon.png` (180, dark opaque), `icon-512-maskable.png`, `site.webmanifest`
- App icons: `app-icon-dark.png`, `app-icon-red.png` (outline), `app-icon-u.png`
- Lockups: `unihub-lockup-stacked(.-dark).png`, `unihub-lockup-horizontal(-dark).png`
- `og-image.png` (1200×630), `pattern-tile.png` (512, tileable background element)

## HTML head snippet
```html
<link rel="icon" href="/favicon.ico" sizes="48x48">
<link rel="icon" href="/unihub-mark-32.png" type="image/png" sizes="32x32">
<link rel="apple-touch-icon" href="/apple-touch-icon.png">
<link rel="manifest" href="/site.webmanifest">
<meta name="theme-color" content="#1a1a1a">
<meta property="og:image" content="https://unihub.live/og-image.png">
<meta property="og:image:width" content="1200">
<meta property="og:image:height" content="630">
<meta name="twitter:card" content="summary_large_image">
```
